Slimecoin.io games are competitive arcade sessions with protocol-backed scoring and settlement.

## Modes

| Mode | Player experience | Protocol path |
| --- | --- | --- |
| Free play | Play without a USDC buy-in | Uses the same game/session rules with a 0 buy-in |
| SLIME queue | Play using SLIME | Reserves SLIME and settles SLIME winnings |
| Paid queue | Play with USDC stakes | Reserves USDC, settles winner payout, mines Slimecoin, and funds fees |

## Current library

The current on-chain library includes:

- Asteroids Blitz
- Farm Wars
- Slime Solitaire
- Block Puzzle
- Blob Battle
- 2048
- Bubble Shooter
- Match 3
- 21 Solitaire

Each game has its own scoring model, but all games share the same basic settlement idea: play a deterministic session, submit or resolve a score, and let the vault credit outcomes.

For game IDs, scoring constants, session windows, and settlement details, see the [game library](/games/library).
